Neovim
I use neovim to code this website, i think it is the best text editor ever made. Because it is lightweight and it is the only thing that makes me code as fast as i can think. The waiting time in some other text editors and IDE's is sometimes too long for me. But there is always a downside and that is if you are trying a new language installing and configuring a new lsp is not always that good and sometimes it doens't work. I am also trying to learn Emacs to see if it actually better, but in the mean time this is the best i have ever seen. Also if i use a different text editor or ide. I always install the vim motions plugin.
